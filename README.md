Esse projeto coleta dados meteorológicos e criminais no período de 2021 a 2026, limpa os DataFrames e filtra apenas para o município de Passo Fundo. Também é feito a padronização das variáveis e junção dos DataFrames, após isso é removido os valores ausentes e corrigido inconsistências nas variáveis categóricas. Por último foi convertido variáveis categóricas em numéricas, identificado e tratado outliers e aplicado normalização. Geramos 3 gráficos com esse DataFrame normalizado final: Distribuição de temperatura máxima, crimes por dia em Passo Fundo e Crimes x Precipitação.
